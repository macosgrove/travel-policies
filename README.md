# Travel Policies

This app provides travellers with quotes for their travel insurance policies.

## Getting started

You'll need to set up the DB:

```shell
rails db:migrate
```

## Run the system tests

```shell
rails test:system
```

## Run the unit tests

```shell
rails test
```

## Run rubocop

```shell
rubocop
```

## Run rubocop and all tests

```shell
./bin/test
```

## Run the app locally

```shell
rails server
```

Then browse to http://localhost:3000
